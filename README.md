# Adaptive UI using AI (HCI Project)

## 📌 Overview
This project demonstrates an Adaptive User Interface (AUI) that dynamically changes based on user interaction patterns.

## 🧠 Features
- Tracks user clicks
- Reorders UI elements based on usage
- Highlights most frequently used feature
- Uses AI-inspired adaptive logic

## 🚀 How it Works
- User clicks buttons
- System stores interaction in localStorage
- UI updates dynamically
- Most used feature moves to top

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Browser localStorage

## ▶️ How to Run
1. Download the project
2. Open `index.html` in browser
3. Click buttons multiple times
4. Observe adaptive behavior
